Linux logrotate е‘Ҫд»ӨиҜҰи§Ј

logrotate жҳҜ Linux зі»з»ҹдёӯз”ЁдәҺиҮӘеҠЁз®ЎзҗҶж—Ҙеҝ—ж–Үд»¶иҪ®иҪ¬гҖҒеҺӢзј©гҖҒеҲ йҷӨе’ҢеҪ’жЎЈзҡ„е·Ҙе…·пјҢйҳІжӯўж—Ҙеҝ—ж–Үд»¶ж— йҷҗеўһй•ҝеҚ ж»ЎзЈҒзӣҳз©әй—ҙгҖӮ

дёҖгҖҒж ёеҝғжҰӮеҝө

1. е·ҘдҪңеҺҹзҗҶ

  • е®ҡжңҹжү§иЎҢпјҡйҖҡиҝҮ cron е®ҡж—¶д»»еҠЎпјҲйҖҡеёёжҜҸеӨ©жү§иЎҢдёҖж¬Ўпјү
  • й…ҚзҪ®й©ұеҠЁпјҡиҜ»еҸ–й…ҚзҪ®ж–Үд»¶еҶіе®ҡеҰӮдҪ•еӨ„зҗҶж—Ҙеҝ—
  • зҠ¶жҖҒи®°еҪ•пјҡи®°еҪ•дёҠж¬ЎиҪ®иҪ¬ж—¶й—ҙеҲ°зҠ¶жҖҒж–Үд»¶

2. е…ёеһӢеә”з”ЁеңәжҷҜ

  • Web жңҚеҠЎеҷЁж—Ҙеҝ—пјҲNginx/Apacheпјү
  • еә”з”Ёж—Ҙеҝ—пјҲJava/Python еә”з”Ёпјү
  • зі»з»ҹж—Ҙеҝ—пјҲsyslog/rsyslogпјү
  • ж•°жҚ®еә“ж—Ҙеҝ—пјҲMySQL/PostgreSQLпјү

  • жҜҸеӨ©иҪ®иҪ¬пјҢдҝқз•ҷ 14 еӨ©

  • еҺӢзј©еҗҺеҸҳжҲҗпјҡaccess.log-20240114.gz

йҖӮз”ЁеңәжҷҜпјҡеә”з”ЁжҢҒз»ӯеҶҷе…Ҙж—Ҙеҝ—ж–Үд»¶пјҢж— жі•дјҳйӣ…йҮҚеҗҜ

  1. йҒҝе…ҚеңЁй«ҳеі°жңҹжү§иЎҢпјҡи°ғж•ҙ cron ж—¶й—ҙеҲ°еҮҢжҷЁ 3-4 зӮ№
  2. еҲҶж•ЈиҪ®иҪ¬ж—¶й—ҙпјҡдёҚеҗҢеә”з”Ёй…ҚзҪ®еңЁдёҚеҗҢж—¶й—ҙж®ө
  3. **дҪҝз”Ё delaycompress**пјҡеҮҸе°‘ I/O еҺӢеҠӣ
  4. е®ҡжңҹжё…зҗҶж—§ж—Ҙеҝ—пјҡз»“еҗҲ maxage жҲ–зӢ¬з«Ӣи„ҡжң¬

  1. ж—Ҙеҝ—ж•Ҹж„ҹдҝЎжҒҜеӨ„зҗҶпјҡ

жҖ»з»“

з»ҙеәҰ е…ій”®зӮ№
ж ёеҝғеҠҹиғҪ иҮӘеҠЁиҪ®иҪ¬гҖҒеҺӢзј©гҖҒеҲ йҷӨгҖҒеҪ’жЎЈ
жү§иЎҢж–№ејҸ Cron е®ҡж—¶д»»еҠЎ + й…ҚзҪ®ж–Үд»¶й©ұеҠЁ
йҮҚзӮ№еҸӮж•° rotateгҖҒcompressгҖҒpostrotateгҖҒcopytruncate
и°ғиҜ•е‘Ҫд»Ө logrotate -d -v
еёёи§Ғйҷ·йҳұ ж–Үд»¶еҸҘжҹ„жңӘйҮҠж”ҫгҖҒжқғйҷҗй”ҷиҜҜгҖҒж—¶й—ҙеҲӨж–ӯеӨұж•Ҳ

жңҖдҪіе®һи·өпјҡ

  • дјҳе…ҲдҪҝз”Ёеә”з”ЁиҮӘеёҰж—Ҙеҝ—иҪ®иҪ¬пјҲеҰӮ Nginx зҡ„ reopen дҝЎеҸ·пјү
  • жөӢиҜ•зҺҜеўғйӘҢиҜҒй…ҚзҪ®еҗҺеҶҚдёҠзәҝ
  • зӣ‘жҺ§зЈҒзӣҳз©әй—ҙе’ҢиҪ®иҪ¬жү§иЎҢзҠ¶жҖҒ
  • е®ҡжңҹе®ЎжҹҘж—Ҙеҝ—дҝқз•ҷзӯ–з•ҘжҳҜеҗҰз¬ҰеҗҲеҗҲи§„иҰҒжұӮ
