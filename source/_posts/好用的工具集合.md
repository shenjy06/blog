## 产品工具

### 需求选择

- 团队协作 / 多人编辑 → 推荐 [Figma](https://www.figma.com/)、[Adobe XD](https://adobexdplatform.com/)、[Marvel](https://marvelapp.com/)
- 快速验证产品想法 / 用户流程 → [Balsamiq](https://balsamiq.com/)、[Mockplus](https://www.mockplus.com/)
- 复杂交互逻辑 / 产品流程多分支 → [Axure RP](https://www.axure.com/)
- 需要高保真 / 动画 / 视觉设计 → [Sketch](https://www.sketch.com/) / [Framer](https://www.framer.com/) / Adobe XD
- 跨平台 / 云端 / 不依赖安装 → Figma、Marvel

### 工具适合的场景

- 快速出雏形／草图 → Balsamiq、Mockplus
- 界面设计 + 简单交互 + 协作 → Figma、Adobe XD、Marvel
- 复杂逻辑／动态交互／流程驱动产品 → Axure RP、Framer
- 视觉精细设计 + macOS 用户 → Sketch
- 动画 / 高保真动效演示 → Framer、Adobe XD（基础动效）
