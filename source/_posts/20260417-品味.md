## 品味

Linus Torvalds 说过："有时候你可以从不同角度看问题，重写它让特殊情况消失，变成正常情况。"这就是品味——不是做更多，而是让对的变得显而易见。

品味无法速成，但可以通过**大量看、反复用、持续打磨**来积累。以下是一些在代码和设计中反复被印证的原则：

| 原则 | 说明 |
|------|------|
| 消除，而非添加 | 好设计是减法。删除直到无法再删，剩下的才是精华。 |
| 一致性 > 花样 | 一个地方的圆角 4px，另一个地方 8px——这不是创意，是 bug。 |
| 空白是奢侈品 | 留白不是浪费，是给内容呼吸的空间。 |
| 层级决定一切 | 用户扫读，不是阅读。字号、字重、颜色三条线撑起信息层级。 |

### 灵感搜集

灵感是品味的养料。多看最好的作品，审美线自然会抬升。

- [Dribbble](https://dribbble.com/) — 设计师社区，UI/插画/动效的首选灵感来源
- [Behance](https://www.behance.net/) — Adobe 旗下，完整作品案例展示
- [Awwwards](https://www.awwwards.com/) — 全球最佳网站评选，前沿 Web 设计趋势
- [Mobbin](https://mobbin.com/) — 真实 App 截图库，竞品分析与交互参考
- [Screenlane](https://screenlane.com/) — 移动端 UI 截图收集，按页面类型分类
- [Collect UI](https://collectui.com/) — 按组件分类的 UI 灵感库（404页/登录页/表单等）
- [Land-book](https://land-book.com/) — 精选 Landing Page 设计集
- [Muzli](https://muz.li/) — Chrome 插件，每次新标签页推送设计灵感
- [Pinterest](https://www.pinterest.com/) — 图片灵感搜索，适合 Mood Board
- [Siteinspire](https://www.siteinspire.com/) — 极简风格网站收藏
